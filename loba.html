Loba - Pill Management App

UX Integration

UX Design, Mobile

About The Brand

Loba is a health and wellness app designed to help users manage their medications, track their health, and stay on top of their well-being. It aims to simplify the process of medication management, ensuring users can easily stay on track with their prescriptions and overall wellness routine.

The Challenge:

For this project, we were tasked with integrating an "Order/Refill Pill" feature into Loba, an existing health app. The challenge was to seamlessly embed this functionality within the app’s current design while maintaining the current user experience. Included are the following mandatories:

  1. Adding a barcode scanner in the Add a Pill page
  2. Notifications when pills are running low
  3. Brand name for pills must be included

CURRENTS SCREENS

Proposed User Flow

Main Objective:

Seamlessly add a refill feature and barcode scanning without disrupting the existing user experience.

Implementation:

  1. Refill Reminders – A proactive “Refill now” prompt on the Home Page helps users refill medication before they run out.
  2. Low-Stock Alerts – The “My Pills” section now detects when medication is running low and offers a quick reorder option.
  3. Barcode Scanning – Added as an option to add new medications, in addition to manual entry.

WIREFRAMES

To enhance usability, I made minimal but impactful UI refinements, including simplified icons, improved card layouts, and better text hierarchy to create visual prominence where needed. These adjustments ensure a cleaner, more intuitive experience while preserving the app’s original design language.
